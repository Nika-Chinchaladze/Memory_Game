How To Play:

1. Player has to find all hidden Symbols step-by-step.

2. Player can't demand next Question until he/she hasn't found asked Symbol.

3. Time is Unlimited, so Player can think as much as he/she wants.

4. There are different categories, based on mistakes Quantity, for example:

  • Less than 5, FIRST Class Player - Memory Strength is PERFECT.
  • Between 6 and 10, SECOND Class Player - Memory Strength is GOOD.
  • Between 11 and 15, THIRD Class Player - Memory Strength is NORMAL.
  • More than 15, FOURTH Class Player - Memory Strength is WEAK.
